Font Awesome полностью бесплатен и поддерживает GPL. Вы можете использовать его в коммерческих проектах, в бесплатных проектах, да и везде, где только пожелаете.

Лицензия на шрифт

  • Применяется ко все настольным и веб-шрифтам, находящимся в папке: font-awesome/fonts/.
  • Лицензия: {{ site.fontawesome.license.font.version }}
  • Описание: {{ site.fontawesome.license.font.url }}

Лицензия на код

  • Применяется ко всем CSS и LESS файлам, находящихся в папках: font-awesome/css/, font-awesome/less/, и font-awesome/scss/.
  • Лицензия: {{ site.fontawesome.license.code.version }}
  • Описание: {{ site.fontawesome.license.code.url }}

Лицензия на документацию

  • Применяется ко всем проектам Font Awesome, которые не относятся ни к разделу шрифтов, ни к разделу кода.
  • Лицензия: {{ site.fontawesome.license.documentation.version }}
  • Описание: {{ site.fontawesome.license.documentation.url }}
